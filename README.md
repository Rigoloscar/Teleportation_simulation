# Teleportation_simulation
--------------------------
This is the first version of the quantum teleportation code for CTTC. This version has, as by now, three files:

- SPDC_source_module.py: this module has both the functions needed to simulate SPDC type II (temporal and spatial walk-off, intensity plots, etc) and the function to
simulate the entanglement source, which is independent from the others and can be called by just knowing the brightness and temporal walk-off of the source without
doing the other simulation.
- Optical_module.py: this module has all the operators of the optical components used (HWP, QWP, LP, BS and PBS).
